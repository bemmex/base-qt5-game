# base-qt5-game
## QT5 Game Template

Edit the CMakeLists file, with the project name and QT libraries.
Run cmake-gui, select the project folder and the build folder
press the Configure button and then Generate button.
Run the commands "cd build" and then "make" to compile and generate the executable.

Alternatively you can run the following commands:
```bash
cd build
cmake ..
make
```
